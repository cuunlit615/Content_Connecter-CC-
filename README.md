# Content_Connecter-CC-
Connect tags content in blocks by line
指令集:
CC:複製線段左側內容到右側
  1.點選頭尾有連接圖塊的線段，並自動同步TAG內容
  2.左右位置(X軸數值)相同時進入CCC模式
CC_SET:CC功能設定
  1.端點誤差容許值(預設0.5單位)
  2.設定TAG1名稱(預設M00)
  3.設定TAG2名稱(預設22/2)
  4.設定TAG3名稱(預設PHNX)
CC_RESET:恢復預設值

CCC:複製Block內的Tag內容
  1.點選來源圖塊
  2.點選目標圖塊
CCC_SET:設定指定Tag名稱
  1.設定TAG1名稱(預設M00)
  2.設定TAG2名稱(預設22/2)
  3.設定TAG3名稱(預設PHNX)
CCC_RESET:恢復預設值
